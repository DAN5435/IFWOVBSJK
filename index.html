LED Control Panel

Click the buttons to control the virtual LED light

LED: OFF

How This Works:

  1. The HTML creates the structure (buttons, LED display, text)
  2. CSS styles everything to look like a real control panel
  3. JavaScript adds interactivity:
    • Clicking "TURN ON" lights up the LED
    • Clicking "TURN OFF" turns off the LED
    • Clicking "TOGGLE" switches between on/off states
